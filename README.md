      <th width="20%">Decision</th>
      <th width="50%">Rationale</th>
      <th width="30%">Impact</th>
    </tr>
    <tr>
      <td><b>ECS Fargate</b> over EC2</td>
      <td>Unpredictable traffic spikes during government announcements. Scales 2 to 20 containers in under 60 seconds without managing instance fleets.</td>
      <td>Eliminated OOM crashes during high-traffic periods</td>
    </tr>
    <tr>
      <td><b>CloudFront + S3</b> for static assets</td>
      <td>Serving through edge nodes reduces Time to First Byte for users outside the capital.</td>
      <td>80% reduction in frontend load time for regional offices</td>
    </tr>
    <tr>
      <td><b>RDS Aurora</b> over self-managed PostgreSQL</td>
      <td>Government requires point-in-time recovery and automated backups with compliance-grade 35-day retention.</td>
