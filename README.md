## About

Full-stack software engineer based in Addis Ababa, Ethiopia. I build software that operates under real conditions — offline environments, government infrastructure, enterprise workflows. My work spans full-stack product development and AWS-based DevOps, with a track record of shipping production systems used by thousands of users.

I study Computer Science / Software Engineering at Addis Ababa University. I approach engineering as a systems discipline: every application is designed to operate reliably, scale predictably, and be maintainable long-term.

## AWS Architecture (Case Study: CivicVoice)

<table>
  <tr>
    <th>Decision</th>
    <th>Rationale</th>
    <th>Impact</th>
  </tr>
  <tr>
    <td><b>ECS Fargate</b> over raw EC2</td>
    <td>Unpredictable traffic spikes when government announcements go out. ECS scales from 2 to 20 containers in under 60 seconds without managing instance fleets.</td>
    <td>Eliminated out-of-memory crashes during high-traffic periods</td>
  </tr>
  <tr>
    <td><b>CloudFront + S3</b> for static assets</td>
    <td>Serving the React frontend through CloudFront edge nodes reduces Time to First Byte for regional users.</td>
    <td>80% reduction in frontend load time for regional government offices</td>
  </tr>
  <tr>
    <td><b>RDS Aurora</b> over self-managed PostgreSQL</td>
    <td>Government data requires point-in-time recovery and automated backups with compliance-grade retention.</td>
    <td>Zero backup management overhead. RPO of 5 minutes</td>
  </tr>
</table>
